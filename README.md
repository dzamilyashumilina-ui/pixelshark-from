Strona internetowa poświęcona serialowi "From", stworzona w ramach nauki tworzenia stron i aplikacji internetowych 
Projekt zawiera sekcje Hero, opisy postaci oraz sekcję FAQ.
FAQ w którym są najczęściej zadawane pytanie.
Były zrobione dodatkowe sekcję,strona zawiera interaktywne elementy.
Całość została stworzona z wykorzystaniem kodu HTML, CSS oraz JavaScript.
Czego się nauczyłam? Git i GitHub: 
Praktyczna obsługa systemu kontroli wersji, tworzenie commitów, zarządzanie gałęziami (master/main) oraz publikacja kodu online.
CSS Grid i Flexbox: 
Tworzenie responsywnych układów stron, które poprawnie wyświetlają się na komputerach i telefonach. Z czym miałam trudność?
Konfiguracja Git:
Napotkałam trudności z autoryzacją konta GitHub w terminalu oraz z poprawnym podpięciem adresu repozytorium (remote origin).
Zarządzanie gałęziami:
Zrozumienie różnicy między gałęzią master a main podczas pierwszego wypychania kodu (push).vv
